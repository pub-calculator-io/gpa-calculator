Calculate your high school or college GPA in seconds. Our free GPA calculator computes your cumulative Grade Point Average using letter or number grades.

[https://www.calculator.io/gpa-calculator/](https://www.calculator.io/gpa-calculator/)

== Usage ==

Add the GPA Calculator shortcode to your page, post or sidebar:

`[calcio_gpa_calculator]`

Add the following code to your template where you would like the GPA Calculator to appear:

`<?php display_calcio_gpa_calculator(); ?>`

== Installation ==

1. Upload the GPA Calculator /calcio_gpa_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the GPA Calculator plugin through the "Plugins" menu in WordPress.
